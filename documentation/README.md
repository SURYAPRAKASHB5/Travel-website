# Apex Voyages - Premium Travel Homepage

An elegant, high-performance, and responsive premium travel homepage built from the ground up utilizing semantic HTML5, modern CSS3 layout systems, and modular Vanilla JavaScript. This project is meticulously crafted to mimic the visual rhythm, spacious editorial aesthetics, and immersive user experience of top-tier travel platforms like Wilderness Travel, completely independent of external dependencies or frameworks.

---

## 📁 Project Folder Structure

The project is structured logically into modular files to demonstrate production-grade architectural organization, ease of maintenance, and separation of concerns:

```text
Travel-Website/
├── index.html
├── css/
│   ├── style.css
│   ├── responsive.css
│   └── animation.css
├── js/
│   ├── script.js
│   ├── slider.js
│   ├── menu.js
│   └── scroll.js
├── images/
└── documentation/
    └── README.md

```
